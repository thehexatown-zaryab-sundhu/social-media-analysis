üîå API Registration & Key Endpoints

üì∏ Instagram API

Registration Process:
  1. Create a Facebook Developer Account at developers.facebook.com
  2. Create a new App and add Instagram Basic Display or Instagram Graph API product
  3. Complete App Review process for advanced permissions
  4. For business features, connect Instagram Business Account
Key API Endpoints:
GET /me?fields=id,username,account_type,media_count - Basic profile info
GET /{user-id}/media?fields=id,caption,media_type,timestamp - User media
GET /{ig-user-id}/insights?metric=follower_count,impressions - Insights (Business only)
GET /{ig-user-id}/insights?metric=audience_gender_age,audience_country - Demographics
Base URL: https://graph.facebook.com/v19.0/

üìò Facebook API

Registration Process:
  1. Create Facebook Developer Account at developers.facebook.com
  2. Create new App and add Facebook Login + Pages API products
  3. Request permissions: pages_read_engagement, pages_show_list, read_insights
  4. Submit for App Review for production use
Key API Endpoints:
GET /{page-id}?fields=name,followers_count,fan_count - Page info
GET /{page-id}/posts?fields=message,created_time,reactions.summary(true) - Posts
GET /{page-id}/insights?metric=page_fans_gender_age,page_fans_country - Demographics
GET /{post-id}/insights?metric=post_reactions_by_type_total - Post insights
Base URL: https://graph.facebook.com/v19.0/

üì∫ YouTube API

Registration Process:
  1. Create Google Cloud Project at console.cloud.google.com
  2. Enable YouTube Data API v3
  3. Create credentials (API Key for public data, OAuth2 for user data)
  4. Set up OAuth consent screen for user authorization
Key API Endpoints:
GET /channels?part=statistics,snippet&id={channel-id} - Channel info & stats
GET /search?part=snippet&channelId={channel-id}&order=date - Channel videos
GET /videos?part=statistics,snippet&id={video-id} - Video details
GET /channels?part=statistics&mine=true - Own channel stats (OAuth required)
Base URL: https://www.googleapis.com/youtube/v3/
Analytics API: https://youtubeanalytics.googleapis.com/v2/

üéµ TikTok API

Registration Process:
  1. Apply at developers.tiktok.com
  2. Submit detailed application with business use case
  3. Wait for approval (can take weeks/months)
  4. Complete developer verification process
  5. Request specific API access (Display API, Marketing API, etc.)
Key API Endpoints:
GET /v2/user/info/?fields=display_name,follower_count,following_count - User info
GET /v2/video/list/?fields=id,title,create_time,view_count - User videos
GET /v2/video/query/?fields=id,title,view_count,like_count - Video details
Research API: https://open.tiktokapis.com/v2/research/ (Academic use only)
Base URL: https://open.tiktokapis.com/

‚ö†Ô∏è Important Considerations:

  • Rate Limits: All platforms have strict rate limiting (Instagram: 200/hour, YouTube: 10,000 units/day, Facebook: varies by endpoint)
  • Authentication: Most valuable data requires user OAuth consent
  • Business Verification: Advanced features often require business verification
  • Compliance: Each platform has specific terms of service for data usage
  • Approval Time: TikTok API approval can take 2-8 weeks
  • Costs: Some APIs have usage fees beyond free tiers

üöÄ Quick Start Recommendations:

  1. Start with YouTube API - Easiest to get approved and most open data access
  2. Facebook/Instagram next - More complex but richest demographic data
  3. TikTok last - Most restrictive, apply early due to long approval times
  4. Use sandbox/test modes initially to understand data structures
  5. Plan for rate limiting in your application architecture
